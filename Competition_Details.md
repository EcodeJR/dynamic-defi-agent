# SuperDapp AI Agents Grants Program — Official Overview

## Program Name
SuperDapp AI Agents Grants Program

## Source
Medium article published Dec 23, 2025 by SuperDapp. :contentReference[oaicite:0]{index=0}

## Grant Structure
- Grants are **performance-based and rolling monthly** — not a fixed hackathon. :contentReference[oaicite:1]{index=1}
- There is **no formal application process**, pitch deck, or deadline. :contentReference[oaicite:2]{index=2}
- Grants scale with **real usage and traction** rather than initial proposals. :contentReference[oaicite:3]{index=3}

## Funding
- Grants are awarded in **$SUPR tokens**. :contentReference[oaicite:4]{index=4}
- Grants are **ongoing monthly** based on engagement. :contentReference[oaicite:5]{index=5}

## Key Judging Criteria
SuperDapp awards grants based on:

1. **Usage** — real user interaction with the agent. :contentReference[oaicite:6]{index=6}
2. **Community Engagement** — active engagement and retention. :contentReference[oaicite:7]{index=7}
3. **Retention** — users come back and use the agent repeatedly. :contentReference[oaicite:8]{index=8}
4. **On-chain Activity** — measurable transactions or state changes triggered by the agent. :contentReference[oaicite:9]{index=9}

## What You Can Build
Agents must be:
- Built using the **SuperDapp SDK**. :contentReference[oaicite:10]{index=10}
- Fully Web3-native and deployable into Super Groups. :contentReference[oaicite:11]{index=11}
- Interactive via commands, APIs, or group chat automation. :contentReference[oaicite:12]{index=12}

Examples include:
- News or content bots
- Community managers
- Campaign or quest agents
- Token-aware logic agents
- Interactive or gamified agents

All must demonstrate **real engagement and usage**, not just prototype demos. :contentReference[oaicite:13]{index=13}

## Eligibility
- Open to **anyone** (solo or teams). :contentReference[oaicite:14]{index=14}
- No formal application or deadlines. :contentReference[oaicite:15]{index=15}
- Grant awards scale with usage, community, and on-chain activity. :contentReference[oaicite:16]{index=16}
